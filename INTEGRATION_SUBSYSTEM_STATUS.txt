6. How to run

Step 1:
Start Docker Desktop.

Step 2:
From this project folder run:

powershell -ExecutionPolicy Bypass -File .\scripts\setup-local-db.ps1

Step 3:
If Docker auto-init does not run the schema for any reason, manually execute:

sql/01-schema.sql

Step 4:
Use the DB config template from:

src/main/resources/application-example.properties

Step 5:
Build again if you make more code changes:

mvn package
